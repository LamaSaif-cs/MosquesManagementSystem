Mosques Management System
A desktop application built with Python and Tkinter designed to manage mosque records efficiently. 
The system allows users to perform core database operations (Create, Read, Update, Delete) through a user-friendly graphical interface.
Features
• Data Entry: Add new mosque records including location and specific details.
• Search Functionality: Quickly find mosque entries by name.
• Database Management: Full support for updating existing records and deleting entries.
• Modern GUI: Organized layout using labels, entry fields, and interactive buttons.
• Error Handling: Robust validation using message boxes to guide the user and prevent system crashes.
Technologies Used
• Language: Python 3.x
• GUI Framework: Tkinter
• Database: SQLite (SQLAlchemy/sqlite3)
• Tools: Standard Messagebox for user alerts.
